# SQL Injection Attack Simulation – DVWA Environment

This repository contains a report and demonstration files for a controlled SQL Injection simulation performed using the **Damn Vulnerable Web Application (DVWA)**.

## 🔍 Objective
To identify and exploit a SQL Injection vulnerability in a safe, educational environment using DVWA.

## 🛠️ Tools Used
- DVWA (hosted on XAMPP)
- Kali Linux (attacker system)
- Browser for web interface

## 📄 Report Includes:
- Introduction to SQL Injection
- Theory Behind the Vulnerability
- Attack Mechanism
- Practical Demonstration using DVWA
- Screenshots of Exploitation Steps
- Conclusion and References

## 📁 Files
- `SQL_Injection_Report.pdf` – Full detailed report
- `/screenshots` – Screenshots from the testing process
- `README.md` – This file

## ⚠️ Disclaimer
This project is strictly for educational purposes. Do not attempt these techniques on live websites or unauthorized systems.
